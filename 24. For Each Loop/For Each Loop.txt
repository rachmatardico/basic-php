For Each
    - Kadang kita bisa mengakses data array menggunakan perulangan.
    - Mengakses data array menggunakan perulangan sangat bertele - tele, 
      kita harus membuat counter, lalu mengakses array menggunakan 
      counter yang kita buat
    - Namun untungnya, di PHP terdapat perulangan for each, yang bisa 
      digunakan untuk mengakses seluruh data array secara otomatis.
