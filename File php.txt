FILE PHP
    - File kode program PHP diakhiri dengan extention .php.
    - Di awal kode program PHP, wajib menambahkan <?php.
      Dan di akhir kode program PHP perlu ditambahkan ?>
      (tapi tidak wajib).
    - Nama file PHP tidak ada aturan, jadi kita bisa membuat 
      file PHP seperti membuat file lain biasanya.
      Namun agar mempermudah saat menjalankan file PHP, 
      direkomendasikan tidak menggunakan spasi.
    - Untuk menampilkan tulisan di PHP, kita bisa 
      menggunakan perintah echo.
