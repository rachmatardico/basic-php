OPERATOR PERBANDINGAN
    - Operator perbandingan, seperti namanya, digunakan untuk 
      membandingkan dua buah value.
    - Hasil dari operator perbandingan adalah boolean, true 
      jika perbandingannya benar, false jika perbandingannya salah.

  Contoh perintahnya : 
    - $a == $b      =>> Sama dengan         =>> True jika $a sama dengan  
                                                sama dengan $b setelah 
                                                dilakukan konversi 
                                                tiap tipe data.
    - $a === $b     =>> Identik             =>> True jika $a sama dengan $b 
                                                dan memiliki tipe data yang 
                                                sama.
    - $a != $b      =>> Tidak Sama Dengan   =>> True jika $a tidak sama dengan 
                                                $b setelah dilakukan konversi 
                                                tipe data.
    - $a <> $b      =>> Tidak Sama Dengan   =>> True jika $a tidak sama dengan 
                                                $b setelah dilakukan konversi 
                                                tipe data.
    - $a !== $b     =>> Tidak identik       =>> True jika $a tidak sama dengan 
                                                $b atau tidak sama tipe data. 
    - $a < $b       =>> Kurang dari         =>> True jika $a kurang dari $b.
    - $a <=$b       =>> Kurang dari atau    =>> True jika $a kurang dari atau 
                        Sama dengan             sama dengan $b.
    - $a > $b       =>> Lebih dari          =>> True jika $a lebih dari $b.  
    - $a >=$b       =>> Lebih dari atau     =>> True jika $a lebih dari atau 
                        Sama dengan             sama dengan $b.
